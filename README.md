Cadillacs & Dinosaurs — Arcade Web Experience
A browser-based arcade gaming page built around the classic 1993 Capcom beat 'em up. The project embeds the game via an iframe emulator and wraps it in a full custom UI featuring character selection, a live score display, and an arcade-themed design.
Supports three play modes: solo offline, local co-op on a shared keyboard with split controls for two players, and online multiplayer via peer-to-peer connections using PeerJS — no server required. Online players connect through a generated room code and can communicate via a built-in room chat.
Built with vanilla HTML, CSS, and JavaScript.
